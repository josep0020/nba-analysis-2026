# NBA Player Statistics Analysis 2026

A data analysis project exploring NBA player performance statistics for the 2026 season using Python and Pandas.

## Overview

This project performs exploratory data analysis on NBA player statistics to identify:
- **Scoring Leaders** - Top point scorers in the league
- **Efficiency Metrics** - Field goal percentage, true shooting metrics
- **Playmaking** - Assist leaders and assist-to-turnover ratios
- **Rebounding & Defense** - Rebounds, steals, blocks analysis

## Dataset

The dataset contains 383 NBA players with 28 features including:
- Player identification (ID, name, team)
- Games played (GP), Minutes (MIN)
- Shooting stats (FGM, FGA, FG_PCT, FG3M, FG3A, FG3_PCT)
- Free throws (FTM, FTA, FT_PCT)
- Rebounds (OREB, DREB, REB)
- Assists (AST), Steals (STL), Blocks (BLK)
- Turnovers (TOV), Personal Fouls (PF)
- Points (PTS), Efficiency (EFF)
