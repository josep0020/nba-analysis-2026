# Análisis de Estadísticas de Jugadores NBA 2026

Un proyecto de análisis de datos que explora las estadísticas de rendimiento de los jugadores de la NBA para la temporada 2026 utilizando Python y Pandas.

## Resumen

Este proyecto realiza un análisis exploratorio de datos sobre las estadísticas de los jugadores de la NBA para identificar:
- **Líderes de Anotación** - Máximos anotadores de la liga.
- **Métricas de Eficiencia** - Porcentaje de tiros de campo, métricas de eficiencia.
- **Creación de Juego** - Líderes en asistencias y ratios de asistencia/pérdida.
- **Rebotes y Defensa** - Análisis de rebotes, robos y tapones.

## Conjunto de Datos

El conjunto de datos contiene 383 jugadores de la NBA con 28 características que incluyen:
- Identificación del jugador (ID, nombre, equipo)
- Partidos jugados (GP), Minutos (MIN)
- Estadísticas de tiro (FGM, FGA, FG_PCT, FG3M, FG3A, FG3_PCT)
- Tiros libres (FTM, FTA, FT_PCT)
- Rebotes (OREB, DREB, REB)
- Asistencias (AST), Robos (STL), Tapones (BLK)
- Pérdidas (TOV), Faltas personales (PF)
- Puntos (PTS), Eficiencia (EFF)
